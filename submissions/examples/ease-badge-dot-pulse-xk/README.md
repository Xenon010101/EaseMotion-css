# Badge Dot Pulse

1. What does this do? A status indicator dot with concentric pulse rings for online, away, and offline states.
2. How is it used? Add a .bdp-dot span inside a .bdp-item; set data-state="online|away|offline" to change color and pulse behavior.
3. Why is it useful? It fits EaseMotion CSS's animation-first philosophy by providing a clear, animated visual indicator for live presence or system status.
