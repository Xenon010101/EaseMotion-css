# Weather Widget Card

A compact weather card showing city name, condition, temperature, and three data points (humidity, wind, UV index). Clicking Refresh randomizes the values within realistic ranges. The temperature is highlighted in indigo.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser. Click Refresh to simulate new weather data.

## Accessibility notes

All data is displayed as text. The Refresh button is a `<button>` element. Values are randomized in ranges that represent plausible weather readings.
