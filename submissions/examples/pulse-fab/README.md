# Pulse Floating Action Button

## 1. What does this do?
A floating action button (FAB) fixed at the bottom-right with a subtle pulse animation to draw attention, stopping on hover.

## 2. How is it used?
Add a `<button>` with the `.pulse-fab` class anywhere in the body. It's fixed-positioned at bottom-right automatically.
```html
<button class="pulse-fab" aria-label="Add">+</button>
```

## 3. Why is it useful?
A ready-to-use FAB with an attention-grabbing pulse effect. The pulse stops on hover for a clean interaction state. Respects `prefers-reduced-motion` and is fully accessible.
