# Loading Spinner

## 1. What does this do?
A simple rotating loading spinner with multiple size and color variants.

## 2. How is it used?
Add a `<div>` with the `.spinner` class. Use `.spinner-sm`, `.spinner-lg`, or `.spinner-accent` for variants.
```html
<div class="spinner" role="status" aria-label="Loading"></div>
```

## 3. Why is it useful?
A lightweight, accessible CSS-only spinner. Multiple sizes and a green accent variant are available via modifier classes. Includes `role="status"` for screen readers and respects `prefers-reduced-motion`.
